How to play

You play with the white pieces. Start the game by clicking the 'Who goes first?' button. You can only move along the lighter squares and only forwards - until you get your piece to the opposite end of the board. There it becomes a 'tower' that can move both ways. You win either by eating all the red pieces or by making it impossible for them to move.

About the game

The game is based on the original code by Kurt Grigg and has been heavily modified by me - Markus Meriluoto. It's built using HTML, CSS and Javascript.
